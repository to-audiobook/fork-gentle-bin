# fork-gentle-bin
prebuilt binaries from fork-gentle

These binaries were compiled with -march=native -mtune=native on the specified machines.

- Colab-CPU
- Colab-T4
- Colab-T4-skylake.avx512
- Kaggle-gpu.T4x2
- Kaggle-no.accelerator

kaldi-models-x.yy.tar.xz is a backup of the kaldi models gentle uses, just in
case something happens to the server they are usually downloaded from.
